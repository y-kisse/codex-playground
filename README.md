# Escape from Tarkov Loot Tracker

このリポジトリでは、FPS ゲーム "Escape from Tarkov" においてレアアイテムを拾った地点を記録するサービスを開発します。Wiki に掲載されている各マップ画像を利用し、UI 上でマップの任意の場所をクリックしてアイテムを選択すると、その場所で取得したアイテムを記録できます。

Web ブラウザ向け UI、Android アプリ、iPhone アプリを提供し、すべて共通のバックエンドサーバーにアクセスする構成とします。

## ディレクトリ構成
- `webui/` - ブラウザで利用する Web UI
- `android/` - Android アプリケーション
- `ios/` - iPhone アプリケーション
- `backend/` - 共通のバックエンドサーバー

## 使い方
1. このリポジトリをクローンします。
2. 各ディレクトリ内の README を参照して環境構築や実行手順を確認してください。
